# 🧠 XQuiz – Command Line Quiz Application (Java OOP)

## 📌 Project Overview

XQuiz is a **command-line based quiz application** developed using **Java and Object-Oriented Programming (OOP) principles**. The application allows users to create quizzes with multiple questions, attempt them, and receive scores based on their answers.

---

## 📅 Project Duration

**Jun 2025**

---

## 🚀 Features

* 📝 Create quizzes with multiple questions
* 🔘 Each question has multiple choices
* ✅ One correct answer per question
* 👤 User can attempt quiz via command line
* 📊 Automatic evaluation and score generation
* 🧪 Unit testing for validating functionality

---

## 🧠 OOP Concepts Used

* **Encapsulation** → Question and Quiz data handled via classes
* **Abstraction** → Core logic separated from user interaction
* **Inheritance** → Reusability across components *(if implemented)*
* **Polymorphism** → Flexible method handling for quiz evaluation

## 📌 How It Works

1. Admin creates quiz with questions and options
2. User attempts quiz via command line
3. System collects answers
4. Application evaluates responses
5. Final score is displayed
